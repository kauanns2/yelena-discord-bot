# Engine

Este diretório contém o núcleo de inteligência da Yelena.

A Engine é responsável por:

- Carregar módulos.
- Ativar sistemas necessários.
- Gerenciar prioridades.
- Controlar memória.
- Construir contexto.
- Planejar respostas.
- Coordenar o raciocínio.
- Entregar a resposta final ao Gemini.

O bot.py apenas envia a mensagem para a Engine.
Toda a inteligência acontece aqui.
